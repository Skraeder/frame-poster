Información sobre el uso de datos de contacto y cookies básicas del sitio.

Volver al inicio

FRAME POSTER utiliza la información enviada en formularios únicamente para fines de contacto, seguimiento de solicitudes y comunicación relacionada con el proyecto. Los datos solicitados pueden incluir nombre, correo electrónico, nombre de marca o estudio, tipo de producto y mensaje.

  • La información no se vende ni se comparte con terceros para fines publicitarios.
  • El formulario de contacto se conecta mediante Formspree para recibir solicitudes.
  • El sitio puede usar cookies básicas para mejorar la experiencia.
  • Si se conecta Shopify, el manejo de datos de compra, pago y checkout dependerá también de la configuración y políticas de Shopify.
  • Para dudas o solicitudes, el usuario podrá contactar a FRAME POSTER por WhatsApp o Instagram.
