Esta carpeta contiene la versión final lista para subir a GitHub y desplegar en Vercel.

- `index.html`

- `script.js`

## Cambios principales

- Se eliminaron de la página pública los analytics internos, CAC, LTV, tasa de conversión, visitas objetivo y dashboard.
- Términos y condiciones están en `terminos.html`.
- Aviso de privacidad está en `privacidad.html`.
- El footer solo enlaza a esas páginas legales.
- Instagram está configurado como `@frame_poster_`.
- WhatsApp está configurado con `55 2089 1454`.
- El QR correcto está en `assets/frame_poster_qr.png`.
- El formulario está conectado con Formspree.

## Formulario

El formulario “Vende con nosotros” está conectado a:

`https://formspree.io/f/xvzykgpd`

Para que llegue el correo:
1. Entra a Formspree.
2. Verifica que el formulario `xvzykgpd` tenga como destinatario el correo correcto.
3. Confirma el correo receptor desde el inbox.
4. Revisa spam/promociones si no llega el primer mensaje.

## Subir a GitHub

1. Descomprime el ZIP.
2. Entra al repositorio `frame-poster`.
3. Sube o reemplaza todos los archivos:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `terminos.html`
   - `privacidad.html`
   - `README.md`
   - carpeta `assets`
4. Haz commit con el mensaje:
   `Subir versión final limpia de Frame Poster`
5. Espera el despliegue automático de Vercel.
6. Prueba:
   - filtros del catálogo
   - links de términos y privacidad
   - WhatsApp
   - Instagram/QR
   - formulario de Formspree
