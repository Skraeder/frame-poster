# FRAME POSTER

FRAME POSTER is an editorial-style eCommerce project focused on curated poster collections, visual storytelling, and a clean online shopping experience.

The website was designed as a digital gallery where users can explore poster collections, view product details, filter by category, and interact with a simple cart experience. The main goal is to combine the aesthetics of an art catalog with the basic functionality of an online store.

## About the Project

This project was developed as part of an eCommerce final assignment. It presents a functional online store concept with a clear brand identity, product strategy, user experience, and digital commerce structure.

FRAME POSTER focuses on creating a modern and visually attractive shopping experience for users interested in posters, design, and editorial-style visual products.

## Main Features

- Editorial-style landing page
- Curated poster collections
- Product catalog with category filters
- Product cards with prices and descriptions
- Shopping cart interaction
- Responsive design for desktop and mobile
- Clean visual identity based on neutral tones, contrast, and minimalist composition

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub
- Vercel

## Project Status

The project is currently presented as an academic eCommerce prototype. Future improvements may include Shopify integration, real checkout functionality, payment gateway setup, inventory management, customer notifications, and order tracking.

## Purpose

The purpose of FRAME POSTER is to demonstrate how a simple online store can be built with a strong visual concept, clear product presentation, and a user-friendly shopping flow.
